# Component State


```javascript
class App extends Component {

  constructor() {

    super();

  

    this.state = {

      name: 'Liam'

    }

  }
```


```javascript
Hi {this.state.name} // Hi Liam
```

* Whenever the variable `this.state.name` changes, React will re-render this
