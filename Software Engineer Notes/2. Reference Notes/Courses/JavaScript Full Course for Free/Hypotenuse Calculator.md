# Hypotenuse Calculator

## Window Prompt

```javascript
let a;
let b;
let c;
 
a = window.prompt("Enter side A:");
a = Number(a); // 3

b = window.prompt("Enter side B:");
b = Number(b); // 4

c = Math.sqrt(Math.pow(a, 2) + Math.pow(b, 2))

console.log("Side C:", c); // Side C: 5
```
